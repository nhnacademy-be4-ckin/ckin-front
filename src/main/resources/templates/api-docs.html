2. 작가

2.1. ID로 작가 조회

주어진 작가 ID로 작가를 조회합니다.

2.1.1. Request

GET /api/authors/1 HTTP/1.1
Host: 133.186.247.149:7030

2.1.2. Response

HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: 50

{
  "authorId" : 1,
  "authorName" : "김작가"
}

2.1.3. Response Field

Path Type Description

authorId

Number

작가의 ID.

authorName

String

작가의 이름.


2.2. 이름으로 작가 검색

작가의 이름을 사용하여 검색합니다. 해당 이름을 포함하는 모든 작가가 결과로 반환됩니다.

2.2.1. Request

GET /api/authors/search?name=%EC%9E%91%EA%B0%80&page=0&size=2 HTTP/1.1
Host: 133.186.247.149:7030

2.2.2. Response

HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: 583

{
  "content" : [ {
    "authorId" : 1,
    "authorName" : "작가1"
  }, {
    "authorId" : 2,
